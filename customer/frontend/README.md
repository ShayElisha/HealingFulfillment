# Customer Service Frontend

שירות Frontend ללקוחות - Healing Fulfillment

## התקנה

```bash
npm install
```

## הרצה

```bash
# Development
npm run dev

# Build
npm run build

# Preview production build
npm run preview
```

## משתני סביבה

צור קובץ `.env`:

```env
VITE_API_URL=http://localhost:5000/api
```

## Port

האפליקציה רצה על פורט 3000
